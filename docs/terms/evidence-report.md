# Term: Evidence Report

## Plain Meaning

An evidence report is a view over stored evidence.

It does not create evidence. It presents evidence matching a scope.

## Common Scopes

Reports may be scoped by:

- one system
- a radius around a system
- one actor
- one corporation
- one alliance
- one region
- a time window

## What A Good Report Shows

A good evidence report states:

- what scope was searched
- what time window was used
- how many killmails matched
- how many activity events matched
- whether the sample is partial
- what provenance exists
- what warnings exist

## What It Must Not Do

An evidence report must not imply certainty beyond the evidence.

Repeated appearances can be called observations or candidate signals. They should not automatically be called proof of staging, ownership, or affiliation.

## Related Layer

Evidence reports belong to the evidence layer.

If a report starts highlighting patterns such as repeated presence or multi-system behavior, that section is moving into observation work product territory and should be labeled carefully.
