# Term: Metadata

## Plain Meaning

Metadata is supporting information that helps Atlas understand or display evidence.

It is not the activity evidence itself.

## Examples

Metadata includes:

- ship names
- ship groups
- system names
- region names
- stargate connections
- current entity names
- current corporation or alliance labels

## Why It Matters

Evidence often contains numeric IDs.

Metadata turns those IDs into readable labels:

```txt
643 -> Armageddon [typeID: 643]
30004660 -> ZTS-4D [solarSystemID: 30004660]
```

## Product Rule

IDs are facts. Names are labels.

Metadata can be refreshed or rebuilt, but it must not replace or rewrite evidence.
