# Deprecated Passive Feed Seed

Status: Deprecated reference
Updated: 2026-05-23

This file was an early passive-feed concept note.

Do not use it as current Atlas doctrine.

Current Atlas behavior is more constrained:

- Atlas does not perform passive broad ingestion.
- Discovery is scoped and explicit.
- Queue refs are possible evidence only.
- ESI expansion is separate and capped.
- Stored evidence comes from expanded ESI killmails.
